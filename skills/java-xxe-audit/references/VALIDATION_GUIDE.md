# 授权复核材料边界

仅确认漏洞或条件成立项可以输出复核请求。待验证、不可确认和非漏洞项不得生成可复制请求。

## 允许

- 使用真实入口、协议方法、参数名和低风险 XML 结构。
- 使用开发单位提供的 canary URL 或无敏感测试文件占位符。
- 明确写“仅限授权测试环境”。
- 不声称已经读取文件、外带成功或验证成功。

## 禁止

- 真实敏感文件路径。
- 内网地址、云元数据地址、外带平台、拒绝服务实体扩展。
- 批量探测或生产环境执行建议。
- 对候选项输出 payload。

## 请求模板

```http
【真实低风险请求；XML 结构需匹配目标对象或消息结构】
```
