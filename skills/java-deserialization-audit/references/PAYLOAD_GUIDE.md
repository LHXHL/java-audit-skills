# 授权复核材料边界

仅确认漏洞或条件成立项可以输出复核材料。待验证、不可确认和非漏洞项不得生成可复制 payload。

## 允许

- 使用真实入口、协议方法、参数名和低破坏测试对象。
- 使用开发单位授权的 canary、无副作用命令或只读环境验证。
- 标注认证状态、环境条件、回滚约束和不能证明的内容。

## 禁止

- 删除、写入、持久化、横向移动、批量利用或业务数据抽取。
- 未授权目标、生产环境执行建议或自动化利用脚本。
- 在入口、decoder 或链条条件未确认时输出 payload。

## 模板

| 项目 | 内容 |
|------|------|
| 入口 | 【真实入口】 |
| payload 类型 | 【低风险验证对象】 |
| 前置条件 | 【授权环境、认证、组件/类型条件、回滚】 |
| 限制 | 【不代表批量利用或生产可执行】 |
