# 反序列化审计报告模板

硬约束：

1. 所有占位符必须替换；无内容写 `无` 或 `不可确认`。
2. 不得写组件版本、CVE、内部规则或 validator 结果。
3. 组件候选不能计入确认漏洞。
4. 只有确认漏洞或条件成立项可包含授权复核材料。
