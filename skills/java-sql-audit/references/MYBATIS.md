# 模板查询机制兼容参考

本文件作为模板查询机制参考；项目专用 trace/helper 应识别实际模板机制。

## 关注点

| 抽象项 | 说明 |
|--------|------|
| `QUERY_TEMPLATE` | 查询模板来源 |
| `VALUE_BINDING` | 值绑定或等价参数化机制 |
| `QUERY_FRAGMENT` | 被直接拼入结构的片段 |
| `DYNAMIC_ORDER_FIELD` | 排序、列名、表名、函数名或方向 |
| `GUARD_CONDITION` | 白名单、枚举、映射或分支限制 |

模板中出现动态片段只是候选；必须结合入口可控性、上下文位置和防护状态。
