# 授权复核材料边界

仅确认漏洞或条件成立项可以输出复核请求。待验证、不可确认和非漏洞项不得生成可复制请求。

## 允许

- 使用真实入口、协议方法、参数名和低风险样例值。
- 使用最小化布尔差异、排序差异、语法差异或短延迟探针。
- 明确写“仅限授权测试环境”。
- 对可能改变数据的路径要求测试库、最小样本、备份或回滚。

## 禁止

- 生产环境执行建议。
- 文件读写、命令执行、外带、批量枚举、业务数据抽取或破坏性 payload。
- 入口未确认时编造请求。
- 对候选项输出可复制请求。

## 请求模板

```http
【真实协议请求；使用低风险值；不要写攻击字符串占位】
```

## payload 说明

| 项目 | 内容 |
|------|------|
| 目标 | 【布尔差异/排序差异/语法差异/短延迟/其他低风险探针】 |
| 适用条件 | 【授权环境、测试数据、回滚约束】 |
| 不能证明 | 【不代表可批量利用或数据可被抽取】 |
