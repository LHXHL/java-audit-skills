# 模块入口输出模板

适用场景：pipeline worker 或模块级路由枚举。

硬约束：

1. 不得列未使用的机制。
2. 不得以 root 或 dispatcher 冒充最终 route。
3. 不得保留占位符。
4. 数量必须精确，并与结构化产物一致。
