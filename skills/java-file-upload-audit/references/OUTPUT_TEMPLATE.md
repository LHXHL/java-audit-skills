# 文件上传/写入审计报告模板

硬约束：

1. 所有占位符必须替换；无内容写 `无` 或 `不可确认`。
2. 不得写可执行 payload、内部规则或 validator 结果。
3. 结论统计必须与映射表状态数量一致。
